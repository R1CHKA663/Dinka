## RTP Система

### Текущие настройки (в базе):
- Dice: 85%
- Mines: 90%
- Bubbles: 95%
- Tower: 88.5%
- Crash: 80%
- X100: 89.5%

### Как работает RTP:
```
win_chance = (RTP / 100) / multiplier
```

**Примеры (Dice, множитель 1.98x):**
- RTP 95% → ~48% шанс выигрыша
- RTP 85% → ~43% шанс выигрыша
- RTP 50% → ~25% шанс выигрыша

### Валидация RTP:
- ✅ Диапазон 10% - 99.9%
- ✅ Валидация в /admin/rtp и /admin/settings
- ✅ Невалидные значения отклоняются с ошибкой

## Вывод средств и Вейджер

### Логика вейджера:
- Вейджер блокирует только **promo_balance**
- **deposit_balance** ВСЕГДА доступен для вывода
- После отыгрыша вейджера - promo_balance (макс 300₽) тоже доступен

### Ручное пополнение:
- ✅ Обновляет `balance` и `deposit_balance`
- ✅ Позволяет выводить сразу после пополнения
- Опция `skip_wager` для пополнения без вейджера

## Промокоды

### Типы:
- **Тип 0**: Фиксированная сумма (например, 100₽)
- **Тип 1**: Процент от депозита (например, 50%)

### Параметры:
- `wager_multiplier`: множитель вейджера (по умолчанию x3)
- `limit`: максимальное количество активаций
- `deposit_required`: требуется ли депозит
